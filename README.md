# MobileSoftware-Project

## 💻 프로젝트 개요
* Kotiln언어 기반으로 앱 개발 프로젝트 수행
* `Final_02_20220760`에서 프로젝트 상세 확인

### #️⃣ 기능 설명
1. 음악 리뷰 등록하기
    * 제목, 가수, 발매일, 장르, 별점을 입력받고 DB에 저장
2. 음악 리뷰 수정하기
    * 수정하기 클릭 시 기존에 저장되어있는 음악에 대한 상세내용 수정하기
    * 수정 내용 DB에 업데이트
